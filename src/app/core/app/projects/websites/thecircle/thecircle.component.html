The Circle

Inspired by "The Circle" by Dave Eggers, our project had three components: a web platform for live streams and chat, an Android app for streaming and chat, and a backend server. Confidentiality was not allowed, so data was stored in plaintext and signed with a private key. RSASSA-PKCS1-v1_5 with SHA-256 was used for verification using the public key. Availability was not a concern, and scalability was a requirement. Our project team of four created the website for the live streams.
