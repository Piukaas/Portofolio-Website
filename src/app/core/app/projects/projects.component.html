2022

The Circle

This project was inspired by "The Circle", a book and film that explores the dangers of transparency and constantly being watched. The project consisted of three components - a website for viewing streams and chatting, an Android app for starting streams, and a backend server for monitoring streams and storing data. The project requirements were that no confidentiality was allowed while keeping integrity and scalability in mind. To maintain data integrity without confidentiality, data was signed using a user's private key and verified using the public key. My responsibility of this project was to create the frontend with three other project members.
