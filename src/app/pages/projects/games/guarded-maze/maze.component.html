Guarded Maze

This project involves creating a game in Unity where the player has to navigate a maze while avoiding obstacles to reach the end goal. The game development process involves following online tutorials to learn the basics of Unity game development, including designing game mechanics, coding, and designing 3D objects. The final product will be an entertaining and challenging game that tests players' problem-solving skills and ability to react quickly in order to avoid the obstacles and successfully navigate through the maze to reach the end goal.
