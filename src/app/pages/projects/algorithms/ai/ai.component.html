AI - Dominoes and Recognition

Appendix C requires creating a Dominoes game where an AI can be added. Three steps are involved: writing a generic domino engine, creating an adversary using a minimax algorithm, and playing without knowing the opponent's tiles. Appendix F requires developing an artificial neural network to recognize handwritten digits using the MNIST dataset. Different network topologies and input parameters are experimented with to see their effect. AUROC is the suggested evaluation function. Parameters include (hidden) layers, perceptrons, and activation function. Results are documented and explained.
