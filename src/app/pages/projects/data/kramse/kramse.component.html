Project Kramse

The task is to convert Kramse's operational database and associated files to a data warehouse using SQL Server. The process involves extracting data from multiple sources, designing a star or snowflake schema, implementing the schema in SQL Server, populating the data warehouse with data, and designing one or more dashboards to meet Kramse's information needs. A product responsibility report detailing the choices made and the reasons behind them is required. The technical files necessary to build the data warehouse must also be included.
