Fysioterapie - Server

This project aims to develop a web application using ASP.net MVC Core, applying a good application architecture and utilizing an automated development pipeline with unit testing and deployment. The project focuses on creating a practice management system for a new physiotherapy practice that Avans University is opening to support students and employees. The system will be used to manage patient records, appointments, and treatment plans. The development team will demonstrate the application and their design choices during the final assessment.
