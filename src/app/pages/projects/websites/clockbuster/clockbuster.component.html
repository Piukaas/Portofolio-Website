Clockbuster - Webdesign

This project involved creating a movie website using HTML5, CSS, JavaScript, and Bootstrap. The goal was to create a visually appealing website that would showcase a variety of movies. Bootstrap was used to ensure that the website was mobile-responsive and user-friendly. The website includes multiple pages for different movie genres and features custom CSS styling to enhance the overall design. The project also involved implementing JavaScript to create interactive elements, such as a search function and movie trailers.
